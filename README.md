# Credit Scoring Model

This project predicts an individual's creditworthiness using Machine Learning classification techniques.

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Features
- Income
- Debt
- Employment Length
- Payment History
- Debt-to-Income Ratio

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

## Result
Random Forest achieved 99% accuracy on the test dataset.
